# Icons folder

All icons on the site load from here. **No code edits** needed to swap a logo.

```
icons/
  social/     ← GitHub, ArtStation, Itch.io, YouTube, etc.
  ui/         ← Menu, arrows, CV, copy, etc.
```

## Quick replace

1. Open `social/` or `ui/`
2. Replace e.g. `itch.svg` with your custom SVG (same filename)
3. Refresh browser

Icons automatically match site style: they use `currentColor` (inherit link color + hover cyan).

See `social/README.md` for platform icons and SVG export tips.
