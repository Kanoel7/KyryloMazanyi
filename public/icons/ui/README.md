# UI icons

Same system as `social/` — replace `.svg` files to customize.

| File | Used for |
|------|----------|
| `cv.svg` | Download CV |
| `menu.svg` | Mobile menu open |
| `close.svg` | Mobile menu close |
| `copy.svg` | Copy email |
| `check.svg` | Copied confirmation |
| `external.svg` | External / details links |
| `chevron-left.svg` | Project gallery prev |
| `chevron-right.svg` | Project gallery next |

Color and size are applied automatically via CSS mask (see `social/README.md`).
